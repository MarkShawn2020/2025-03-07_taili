## 内容

- 📸 精美的封面展示
- 🎙️ 语音素材资料（即将推出）
- 📝 会议记录与总结
- 🌟 精彩瞬间回顾

## 技术栈

- 🚀 [Next.js 14](https://nextjs.org) App Router
- 🎨 [Tailwind CSS](https://tailwindcss.com)
- 🖼️ 响应式设计
- 🌓 暗色模式支持

## 本地开发

1. 安装依赖
```bash
pnpm install
```

2. 启动开发服务器
```bash
pnpm dev
```

## 项目结构
